# Steam Friends Exporter

Exports your Steam friends list to CSV with one click.

## Features

- One-click export from Steam friends page  
- Excel-safe CSV output  
- Automatically embeds account owner ID  
- Works with vanity (`/id/username`) and numeric (`/profiles/steamid`) profiles  
- Tampermonkey auto-update support  

## Installation

1. Install Tampermonkey extension.
2. Open:

https://github.com/tanersb/Steam-Friends-Exporter/raw/refs/heads/main/steam-friends-export.user.js

3. Click **Install**.

## Usage

1.Go to your Steam friends page  use button 
   Example:steam_friends_TanerSB_4.csv
   
2. Bottom-right panel will appear.

3. Click **EXPORT FRIENDS CSV**.

4. CSV file will be downloaded automatically.

## Output

Filename format:

steam_friends_<AccountID>_<FriendCount>.csv


First column in CSV:

AccountID


This shows which Steam account the export belongs to.
