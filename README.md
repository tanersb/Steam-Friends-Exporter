# Steam Friends Exporter

Exports your Steam friends list to CSV with one click.

## Features

- One-click export
- Excel-safe CSV
- Includes account owner ID
- Works on vanity and numeric profiles
- Auto-update support via Tampermonkey

## Installation

1. Install Tampermonkey extension.
2. Open this link:

https://raw.githubusercontent.com/tanersb/LinkedIn-Fast-Sequential-Auto-Connect/main/steam-friends-export.user.js

3. Click Install.

## Usage

1. Go to your Steam friends page.
2. Bottom-right panel appears.
3. Click **EXPORT FRIENDS CSV**.
4. CSV file is downloaded.

## Output

Filename example:

steam_friends_TanerSB_4.csv
